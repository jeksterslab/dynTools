## Description

Utility functions for data preparation when fitting dynamic models.

## Installation

You can install the released version of `dynTools` from
[GitHub](https://github.com/jeksterslab/dynTools) with:

``` r

if (!require("pak")) install.packages("pak")
pak::pkg_install("jeksterslab/dynTools")
```
